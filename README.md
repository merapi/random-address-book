# Random Address Book

## [Demo](https://random-address-book.netlify.com/)

### Install/run the app

`yarn install`  
`yarn start`

### Testing

`yarn test` - redux-saga tests  
`yarn test:e2e` - cypress e2e tests (after `yarn start`)
